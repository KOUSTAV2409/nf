How it works

~ 
# you found a useful command
$ fuser -k 3000/tcp

# save it before you forget
$ nf "fuser -k 3000/tcp — kills process on port 3000"
Note saved.

# or jot down a thought
$ nf "refactor auth module before friday"
Note saved.

# two weeks later...
$ nf search port
 1  2025-04-25  fuser -k 3000/tcp — kills process on port 3000
 3  2025-04-25  ss -tulpn | grep LISTEN — shows all listening ports

# or just browse everything
$ nf list
 1  2025-04-25  fuser -k 3000/tcp — kills process on port 3000
 2  2025-04-25  refactor auth module before friday
 3  2025-04-25  ss -tulpn | grep LISTEN — shows all listening ports
 4  2025-04-24  tar -xzf archive.tar.gz -C /tmp — extract to /tmp

Commands

nf "text"
save a new note
nf
open TUI (fzf) or list notes
nf list
print all notes
nf search <term>
grep through notes
nf find <term>
alias for search
nf del <n>
delete note by number
nf edit
open notes in $EDITOR
nf count
total number of notes
nf update
update to latest version
nf help
usage
nf version
print version

Why this exists

  • 3 seconds. That's the whole UX. Type nf, paste your note, done.
  • One flat file. Your notes live at ~/.local/share/nf/notes. It's just text. grep it, cat it, rsync it.
  • Zero dependencies. Pure bash. No python, no node, no rust. Runs on any Linux or macOS box.
  • No cloud. Nothing leaves your machine. Ever.
  • fzf optional. If you have fzf installed, nf opens a fuzzy TUI. If not, it just lists notes. Either way it works.

Install

script recommended
$ curl -sL https://nf.iamk.xyz/install | bash
homebrew macos/linux
$ brew install KOUSTAV2409/tap/nf
manual
$ git clone https://github.com/KOUSTAV2409/nf.git
$ cd nf && chmod +x nf.sh
$ sudo ln -s "$(pwd)/nf.sh" /usr/local/bin/nf
then optionally
# for TUI mode (fuzzy search)
$ sudo apt install fzf     # debian/ubuntu
$ sudo pacman -S fzf        # arch
$ sudo dnf install fzf      # fedora
$ brew install fzf           # macOS
