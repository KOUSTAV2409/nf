nf

Note Fast.

A single-command note tool for Linux.
No cloud. No account. Just your terminal.

$ curl -sL https://nf.iamk.xyz/install | bash

How it works

1 Learn a command

you just found this on stack overflow

$ fuser -k 3000/tcp
killed process on port 3000
2 Note it 
$ nf "fuser -k 3000/tcp — kills process on port 3000"
Note saved.
3 Find it later 
$ nf search port
 1  2025-04-25  fuser -k 3000/tcp — kills process on port 3000
 3  2025-04-25  ss -tulpn | grep LISTEN — shows all listening ports

Commands

nf "text" Save a new note
nf Open TUI (requires fzf) or list notes
nf list List all notes
nf search <term> Search notes (case-insensitive)
nf del <n> Delete a note by number
nf edit Open notes in $EDITOR
nf count Show total number of notes
nf help Show help
nf version Show version

Why nf

  • Fast. One command to save. One command to search. Under 3 seconds.
  • Local. No cloud, no account, no internet required. Your notes never leave your machine.
  • Plain text. Notes live in a single file at ~/.local/share/nf/notes. grep it, cat it, back it up.
  • No setup. Install and use immediately. No config files, no databases, no init commands.
  • Open source. MIT licensed. Fork it, extend it, make it yours.

Install

One-liner recommended 
$ curl -sL https://nf.iamk.xyz/install | bash
Manual
$ git clone https://github.com/YOUR_USERNAME/nf.git
$ cd nf
$ chmod +x nf.sh
$ sudo ln -s "$(pwd)/nf.sh" /usr/local/bin/nf
